# 🚀 项目启动指南

> 快速启动泡泡堂游戏的完整步骤

---

## ✅ 环境要求

- **Python 3.7 或更高版本**
- **pip**（Python 包管理器，通常随 Python 安装）

---

## 📦 第一步：安装依赖项

### 方法1：使用 requirements.txt（推荐）

在项目根目录运行：

```bash
pip install -r requirements.txt
```

### 方法2：手动安装

```bash
pip install pygame PyYAML
```

**需要的库：**
- `pygame` - 游戏开发库
- `PyYAML` - 配置文件读取库（用于读取 YAML 配置文件）

---

## 🎮 第二步：启动游戏

### 方法1：使用命令行（推荐）

1. **打开命令行（终端）**

   - **Windows**: 按 `Win + R`，输入 `cmd` 或 `powershell`
   - **Mac/Linux**: 打开 Terminal

2. **进入项目目录**

   ```bash
   cd 2dgame/src
   ```

3. **运行游戏**

   ```bash
   python main2.py
   ```

### 方法2：使用 VS Code

1. **打开项目文件夹**（在 VS Code 中）
2. **打开文件** `2dgame/src/main2.py`
3. **按 `F5`** 或点击运行按钮
   - VS Code 会自动使用 `.vscode/launch.json` 中的配置

### 方法3：直接双击（Windows）

如果 Python 已正确配置：
1. 右键点击 `2dgame/src/main2.py`
2. 选择"用 Python 打开"
3. 游戏窗口会弹出

## 🔧 如果遇到问题

### 问题1：提示找不到 pygame

**错误信息：**
```
ModuleNotFoundError: No module named 'pygame'
```

**解决方法：**
```bash
pip install pygame
```

### 问题2：提示找不到 yaml

**错误信息：**
```
ModuleNotFoundError: No module named 'yaml'
```

**解决方法：**
```bash
pip install PyYAML
```

### 问题3：提示找不到模块

**错误信息：**
```
ModuleNotFoundError: No module named 'player'
```

**解决方法：**
- 确保在 `2dgame/src/` 目录下运行
- 或者在项目根目录运行：`python -m 2dgame.src.main2`

### 问题4：Python 版本太低

**解决方法：**
- 安装 Python 3.7 或更高版本
- 从 [python.org](https://www.python.org/downloads/) 下载

### 问题5：pip 命令不存在

**解决方法：**
- 使用 `python -m pip install pygame PyYAML`
- 或安装 Python 时确保勾选了 "pip"

---

## 📝 验证安装

运行以下命令检查是否安装成功：

```bash
python -c "import pygame; import yaml; print('✓ 所有依赖已安装')"
```

如果看到 "✓ 所有依赖已安装"，说明安装成功！

## 🎯 快速启动脚本（可选）

### Windows 批处理文件

创建 `启动游戏.bat`（在项目根目录）：

```batch
@echo off
cd 2dgame\src
python main2.py
pause
```

双击即可启动游戏。

### Mac/Linux Shell 脚本

创建 `启动游戏.sh`：

```bash
#!/bin/bash
cd 2dgame/src
python3 main2.py
```

然后运行：
```bash
chmod +x 启动游戏.sh
./启动游戏.sh
```

## 🔍 检查项目结构

确保你的项目结构如下：

```
2DGame_py-main/
├── requirements.txt        ← 依赖文件
├── 2dgame/
│   ├── src/               ← 源代码目录
│   │   ├── main2.py       ← 游戏入口（主文件）
│   │   ├── game_manager.py
│   │   ├── player.py
│   │   └── ...
│   ├── config/            ← 配置文件
│   │   └── config.yaml
│   ├── assets/            ← 游戏资源
│   └── map/               ← 地图文件
└── ...
```

---

## 💡 提示

1. **工作目录很重要**
   - 必须在 `2dgame/src/` 目录下运行
   - 或者在项目根目录使用模块方式运行

2. **配置文件路径**
   - 代码中使用相对路径 `../config/config.yaml`
   - 所以必须从 `src/` 目录运行

3. **如果游戏窗口没有弹出**
   - 检查是否有错误信息
   - 查看控制台输出
   - 确保所有资源文件存在

## 🎮 开始游戏

安装完成并成功启动后，你会看到一个游戏窗口：

- **玩家1**：方向键移动，右 Shift 放炸弹
- **玩家2**：WASD 移动，左 Shift 放炸弹
- **数字键 1**：开启/关闭调试模式

详细玩法请查看 `游戏玩法说明.md`
