## 🚀 如何启动游戏

### 方法1：使用批处理文件（最简单）

直接双击项目根目录的 **`启动游戏.bat`**

脚本会自动：
- 检查虚拟环境是否存在
- 如果不存在，自动创建并安装依赖
- 激活虚拟环境
- 启动游戏

### 方法2：使用 VS Code

1. 在 VS Code 中打开项目
2. 按 `F5` 或点击运行按钮
3. 选择 **"启动泡泡堂游戏"**

VS Code 会自动使用虚拟环境中的 Python。

### 方法3：手动命令行启动

```bash
# 1. 激活虚拟环境
venv\Scripts\activate

# 2. 进入游戏目录
cd 2dgame\src

# 3. 运行游戏
python main2.py
```

## 🔧 如果需要重新创建虚拟环境

```bash
# 删除旧的虚拟环境
rmdir /s venv

# 创建新的虚拟环境
py -3.9 -m venv venv

# 激活虚拟环境
venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt
```

---

## ⚠️ 注意事项

1. **不要删除 `venv` 文件夹**
   - 这是你的虚拟环境
   - 删除后需要重新安装依赖

2. **虚拟环境已包含所有依赖**
   - pygame 2.6.1
   - PyYAML 6.0.3
   - Python 3.9.12

3. **VS Code 会自动识别虚拟环境**
   - 如果提示选择解释器，选择 `venv\Scripts\python.exe`

4. **不同项目应该使用不同的虚拟环境**
   - 这样可以避免依赖冲突
